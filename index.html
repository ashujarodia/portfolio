    My Project

    Get Weather App

    GetWeather is a project I created using React JS that allows users to easily access and view weather information for any city. With this application, users can simply type in a city name and get real-time weather data, including temperature, humidity, wind speed, and more. The application uses the OpenWeatherMap API to fetch weather data, and the data is displayed in a user-friendly and visually appealing interface. The application is fully responsive and can be used on desktops, tablets, and mobile devices. Overall, GetWeather is a project that showcases my skills in React JS and my ability to create useful and engaging applications that solve real-world problems.
